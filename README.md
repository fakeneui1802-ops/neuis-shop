# fastapi-starter-de

Ein kleines Starter-Template für eine Python FastAPI-API — fertig für GitHub, Docker und CI.

## Features
- FastAPI als Web-Framework
- Beispiel-Endpoint `/` und `/health`
- Tests mit pytest
- Dockerfile für Container
- GitHub Actions CI (Lint & Tests)
- `Makefile` für einfache Befehle

## Voraussetzungen
- Python 3.10+
- Docker (optional)
- Git
